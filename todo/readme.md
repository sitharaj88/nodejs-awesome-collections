# Todo App 📝

A simple and elegant Todo application built with Node.js, Express.js, and MongoDB. Stay organized, manage your tasks, and boost your productivity with ease.

## Features ✨

- **User Registration and Login** 🚀: Create a personal account or log in to your existing account to access your todo lists.
- **Secure Authentication** 🔒: Safeguard your data with passport-based authentication, ensuring that only authorized users can access and manage their todos.
- **Create, Read, Update, and Delete Todos** 📋: Effortlessly create new todos, keep track of your tasks, update their status, and delete completed or irrelevant todos.
- **User-Specific Todo Lists** 🌟: Each user has their own personalized todo list, enabling a private and customized experience.
- **Responsive and Intuitive Interface** 🎉: Enjoy a sleek and user-friendly interface that adapts seamlessly to different screen sizes and devices.

## Technologies Used 💻

- **Node.js** 🟢: A powerful JavaScript runtime that allows us to build fast and scalable applications.
- **Express.js** 🌐: A minimalist web framework for Node.js that provides a robust set of features for web and API development.
- **MongoDB** 🍃: A flexible and scalable NoSQL database used for efficient storage and retrieval of user data and todos.
- **Passport.js** 🔑: A popular authentication middleware for Node.js that simplifies user authentication and session management.
- **EJS (Embedded JavaScript)** 🎨: A templating engine that enables dynamic content rendering and easy integration of server-side code in HTML templates.
- **Bootstrap** 🎉: A front-end framework that provides ready-to-use CSS styles and components for a responsive and visually appealing design.

## Getting Started 🏁

Follow the steps below to get the Todo app up and running on your local machine:

### Prerequisites 📋

- **Node.js** (v12 or above) should be installed on your machine.
- **MongoDB** should be installed and running.

### Installation ⚙️

1. Clone the repository:

```shell
git clone https://github.com/your-username/todo-app.git
```

2. Navigate to the project directory:

```shell
cd todo-app
```

3. Install the dependencies:

```shell
npm install
```

4. Set up the environment variables:
   - Rename `.env.example` file to `.env`.
   - Update the MongoDB connection URL in the `.env` file.

5. Start the application:

```shell
npm start
```

6. Open your web browser and visit `http://localhost:3000` to access the Todo app.
