# 📝 Todo App

Manage your todos with ease! This simple and intuitive Todo App helps you stay organized and focused.

✨ **Please star this repository if you find it useful!** ✨

## ✅ Features

🔒 **User Authentication:** Securely register and log in to your account.  
📋 **Todo Management:** Create, view, edit, and delete your todos.  
👤 **User-Specific Data:** Each user has their own personalized todo list.  
🎨 **Beautiful UI:** Enjoy a clean and intuitive user interface.  
🌐 **Responsive Design:** Access and manage your todos on any device.

## 🚀 Technologies Used

- Node.js
- Express.js
- MongoDB
- Passport.js
- Bootstrap
- EJS

## ⚙️ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sitharaj88/todo-app.git
   ```

2. Navigate to the project directory:
   ```bash
   cd todo-app
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm start
   ```

5. Open your web browser and visit `http://localhost:3000`.
